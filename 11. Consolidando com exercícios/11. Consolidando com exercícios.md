## 11.1 - Sistema de notas escolares
    Crie um alritmo que transforme as notas do sistema numérico para sistema de notas em caracteres tipo A B C

    * de 90 para cima   -     A
    * entre 80 - 89     -     B
    * entre 70 - 79     -     C
    * entre 60 - 69     -     D
    * menor que 60      -     F
